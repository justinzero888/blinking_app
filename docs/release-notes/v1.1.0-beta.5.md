## What's New

### 🎉 7-Day Free AI Trial (PROP-6)

Zero-configuration AI access for new users. No API key needed — just tap and chat.

- **"Start Free Trial →"** banner in Settings → AI Provider
- **7 days** of unlimited access to the AI assistant with **20 requests/day**
- **Powered by qwen3.5-flash** via OpenRouter — fast, capable, and cost-effective
- **Trial provider** appears in the provider list with remaining days
- **Auto-fallback:** AI assistant uses trial when no user API key is configured
- **Smooth transition:** When trial expires, users are guided to add their own API key
- **Floating robot** shows animated when trial is active, grey clock badge when expired

### 🔧 Backend

- New Cloudflare Worker endpoints at `blinkingchorus.com/api/trial/start` and `/api/trial/chat`
- Secure anonymous device identification (no personal data sent)
- Server-enforced rate limiting and expiry
- Kill switch for emergency disable without redeployment

### 🌐 Localization

- 13 new EN/ZH strings for all trial UI states
- Trial banner, provider entry, info dialog, error messages all bilingual

---

## Technical

- **Architecture:** Cloudflare Worker + KV → OpenRouter proxy with rate limiting
- **Model:** `qwen/qwen3.5-flash` (~$0.00005/request, negligible cost at beta scale)
- **Security:** Opaque token authentication, anonymous device IDs, server-authoritative clock
- **Tests:** 94 Flutter tests + 343 backend tests (all passing)
- **Storage:** KV key-value store for trial state (no D1 migration needed)

## Known Limitations

- Trial requires internet connectivity (server-enforced)
- Changing device clock does not extend trial (server's clock is authoritative)
- One trial per device — reinstalling the app after clearing data does NOT reset trial
- iOS not yet supported (blocked on Xcode 26 / Flutter compatibility)
