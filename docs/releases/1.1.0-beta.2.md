# Blinking — Release Notes
## Version 1.1.0-beta.2 (Build 10)
**Date:** 2026-03-25
**Track:** Google Play Internal Testing

---

## What's New

### App Icon
- Redesigned app icon now fills the full launcher circle on Android 8+ using adaptive icon technology — consistent with the visual style of other system apps.

### Memory Card — Major Improvements

**Custom Background Image**
You can now set a personal photo or image as the background of any card template. The custom background is correctly applied to the generated card image and preserved when sharing — exactly what you designed.

**Template Names in English**
When your device language is set to English, card template names now display in English (Spring Day, Midnight Blue, Warm Sunrise, Minimal, Forest Green, Custom).

**Smart Text Sizing**
Card text now automatically scales to fill the card. A short quote becomes a bold, large statement. A longer passage uses a smaller, elegant size. The text always fills the dedicated text area — no awkward empty space.

**Share Image Only**
Sharing a card now sends the card image by itself — clean and ready to post. No more duplicate text alongside the image.
