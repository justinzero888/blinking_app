# Issues #4, #7, #14: Combined UAT — User Acceptance Testing

**Build:** v1.1.0-beta.5+20 | **Date:** 2026-05-01 | **Tester:** _______________ | **Result:** PASS / FAIL

---

## What Changed

Three UX fixes implemented in one batch:
- **#14:** HomeScreen title and bottom nav tab renamed from "Calendar"/"日历" → "My Day"/"我的一天". Title shows date context (e.g. "My Day - Apr 15" when viewing past dates, just "My Day" for today).
- **#4:** AI Secrets lock icon — small grey `Icons.lock_outline` appears on entry cards and entry detail screen when entry has the "Secrets" tag.
- **#7:** Contextual FAB — on Routine tab the FAB icon changes to `Icons.playlist_add` and opens the Add Routine dialog. The redundant AppBar add button on Routine tab is removed.

---

## Setup

- [ ] App installed on Android emulator and opens without crash
- [ ] At least one entry has the "Secrets" tag applied (Settings → Tags → Secrets or apply in entry editor)
- [ ] At least one routine exists for the Routine tab test
